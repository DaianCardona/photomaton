# Glosario

Términos usados de forma consistente en toda la documentación.

| Término | Definición |
|---------|------------|
| **Evento** | Boda, bautizo, comunión, fiesta o evento de empresa para el que se contrata el fotomatón. Genera una carpeta propia. |
| **Sesión / toma** | Conjunto de fotos disparadas por un grupo de invitados en una interacción (1–4 fotos). |
| **Plantilla (template)** | Diseño a tamaño de papel (10x15 o 5x15 cm) con huecos para las fotos y elementos gráficos (texto, logos, decoración). |
| **Layout** | Resultado final montado: las fotos de la sesión colocadas dentro de la plantilla, listo para imprimir. |
| **Original** | Foto tal cual sale de la cámara (JPG/RAW), sin filtros ni montaje. Nunca se modifica. |
| **Single** | Cada foto individual ya recortada/procesada (filtro, fondo) pero fuera de la plantilla. |
| **Tira** | Formato vertical 5x15 cm (media hoja). Una hoja 10x15 puede cortarse en 2 tiras 5x15. |
| **Corte (cut)** | Función de la impresora DNP que parte la copia 10x15 en dos tiras 5x15. Configurable. |
| **Cuenta atrás** | Temporizador en pantalla (3-2-1) antes de cada disparo. |
| **Filtro** | Ajuste de color sobre la foto: normal, blanco y negro, sepia/beige, etc. |
| **Atrezzo virtual** | Adornos digitales superpuestos a la persona (gafas, sombreros, marcos…), estilo filtro de redes. |
| **Atrezzo físico** | Complementos reales que el operador lleva al evento (no gestionados por la app). |
| **Fondo virtual** | Sustitución del fondo real por una imagen. Por **croma** (tela verde) o por **IA** (segmentación sin croma). |
| **Croma / chroma key** | Técnica que reemplaza un color de fondo (verde) por otra imagen. |
| **Confesión** | Vídeo corto con audio que un invitado graba dejando un mensaje al anfitrión. |
| **Live view** | Vídeo en directo de la cámara mostrado en pantalla antes del disparo. |
| **Tethering** | Control de la cámara desde el ordenador por USB (disparo y descarga). |
| **EDSDK** | *Canon EOS Digital SDK*, kit oficial de Canon para controlar réflex por USB. |
| **gphoto2 / libgphoto2** | Librería libre multi-fabricante de control de cámaras; *fallback* del EDSDK. |
| **DNP** | Fabricante de impresoras de sublimación térmica habituales en fotomatón (DS-RX1, DS620, QW410). |
| **CapRover** | PaaS auto-alojado (sobre Docker) donde se despliegan la web, la API y la BBDD de DecaBox. |
| **decabox-bbdd** | Instancia PostgreSQL en CapRover que almacena eventos, fotos y metadatos. |
| **Cola de sync** | Lista de medios pendientes de subir a la web cuando vuelve la conexión. |
| **Galería pública** | Página web por evento donde los invitados ven/descargan sus fotos (acceso por QR). |
| **MVP** | *Minimum Viable Product*: primera versión usable en un evento real. |
| **RF / RNF** | Requisito Funcional / Requisito No Funcional. |
