# Guía de uso en evento

Checklist operativo para Daian/el equipo DecaBox: del montaje al desmontaje. Pensada para imprimir o
consultar en el móvil durante el servicio.

## Antes del evento (en casa/taller)
- [ ] Surface cargada y actualizada (app y SO).
- [ ] Probar cámara (disparo de test) e impresora (impresión de test 10x15 y 5x15 con corte).
- [ ] Cargar/diseñar la **plantilla** del evento (nombres + fecha) en el [editor](../02-app-escritorio/editor-plantillas.md).
- [ ] Definir ajustes del evento: copias, filtro, fondo, atrezzo, idioma ([ajustes](../02-app-escritorio/pantalla-ajustes.md)).
- [ ] Consumible de impresión suficiente para la duración (2h/3h) + repuesto.
- [ ] Espacio en disco suficiente.

## Montaje en el local
- [ ] Colocar estructura/cabina, Surface y cámara; conectar por **hub USB alimentado**.
- [ ] Enchufar todo a regleta con protección; **no** dejar la Surface a batería.
- [ ] Montar **croma** e **iluminación** si se usa fondo virtual.
- [ ] Conectar **micrófono** si hay confesiones.
- [ ] Abrir la app → **crear/abrir el evento** (carpeta `Nombre_DD.MM.AA`).
- [ ] Verificar live view, impresión de prueba y, si hay red, conexión con la API (estado en ajustes).
- [ ] Pasar a **modo invitado**.

## Durante el evento
- [ ] Vigilar la **cola de impresión** y el **papel restante**.
- [ ] Atender reimpresiones desde la [galería](../02-app-escritorio/reimpresion-y-galeria.md).
- [ ] Si no hay red: tranquilidad, todo se guarda y sube luego (offline-first).
- [ ] Ayudar a invitados con QR/email si lo piden.
- [ ] Echar un ojo al espacio en disco y a la temperatura del equipo.

## Cierre / desmontaje
- [ ] Volver a **modo operador** (PIN).
- [ ] Comprobar que la **cola de sync** terminó (si hay red) o dejarla sincronizando.
- [ ] Cerrar/archivar el evento; anotar incidencias.
- [ ] Apagar y recoger hardware con cuidado (cámara, impresora, croma).

## Después del evento (en casa)
- [ ] Conectar a internet y dejar que **termine la sincronización** (fotos, gifs, vídeos, emails).
- [ ] Verificar la **galería web** del evento.
- [ ] **Copia de seguridad** de la carpeta del evento (ver [mantenimiento](mantenimiento-y-backups.md)).
- [ ] Entregar material al cliente si procede; aplicar política de retención.

## Atajos clave
- Entrar a ajustes (modo invitado): gesto/atajo + **PIN**.
- Reimprimir: Galería → seleccionar layout → copias → imprimir.
- Forzar sync: Ajustes → Compartir/conectividad.
